Fabstone CTA button — use for every action; primary red for the main action, ghost on dark/red backgrounds, white on the red CTA banner.

```jsx
const { Button } = window.FabstoneDesignSystem_e3db74;
<Button variant="primary" href="/contact-us/">Contact Us</Button>
<Button variant="ghost">Live Inventory</Button>
```

Variants: `primary` (red, default), `ghost` (transparent + white border — only on dark/red), `dark`, `white` (white fill + red text — on the red banner), `outline-dark` (light pages, secondary). `pill` rounds fully. Label style is Title Case, 1–3 words.
